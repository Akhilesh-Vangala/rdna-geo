# GitHub access

This project is on GitHub as a **private** repository:

**https://github.com/Akhilesh-Vangala/rdna-geo**

## Clone (new machine or collaborator)

```bash
git clone https://github.com/Akhilesh-Vangala/rdna-geo.git
```

You will be prompted for credentials. Use a **Personal Access Token** (not your GitHub password) for HTTPS. Or use SSH:

```bash
git clone git@github.com:Akhilesh-Vangala/rdna-geo.git
```

## Push / pull

From this folder: `git push origin main` or `git pull origin main`. Requires authentication (PAT or SSH key) with access to this repo.

## Getting access

This repo is private. To get access, contact the repository owner to be added as a collaborator (Settings → Collaborators on GitHub).
